# Design Guidelines

- Direction: refined organic minimalism, bright gallery canvas with soft floating bubbles.
- Typography: Be Vietnam Pro for headings, Noto Sans for form copy.
- Primary text: `#172033`; primary accent: `#FF8A3D`; canvas: `#f5f7f2`.
- Use the FPT gradient `linear-gradient(135deg, #F37021, #7AC143, #0E76BC)`
  only on atmospheric background shapes so the form remains calm and legible.
- Keep one primary action, visible labels, 48px minimum controls, and clear status feedback.
- Background motion must use only transform/opacity and stop under `prefers-reduced-motion`.
- Layout is mobile-first and must remain usable at 375px width.
