# Landing Wish Form

## Goal

Build a single-page static form deployable to Vercel. It uploads an image to
Supabase Storage, stores the public URL in a hidden field, then posts all data
to an existing Google Form.

## Scope

1. Responsive animated organic background and accessible form.
2. Drag/drop and click image input with preview and upload feedback.
3. Supabase Storage REST upload and public URL generation.
4. Google Forms submission with configurable field IDs.
5. Setup/deployment documentation.

## Verification

- Validate required fields and Vietnamese phone numbers.
- Validate file type and 5 MB limit.
- Respect reduced-motion.
- Ensure loading, success, and error states are visible.
